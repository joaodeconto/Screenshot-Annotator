Screenshot Annotator • One‑File
Dica: arraste uma imagem, ou cole (Ctrl/⌘+V). Clique e arraste para desenhar. Texto: clique para inserir.

Como publicar (GitHub Pages – 3 passos)

  1. Crie um repositório no GitHub (ex.: screenshot-annotator).
  2. Salve este arquivo como index.html na raiz e faça commit.
  3. Em Settings → Pages, escolha Deploy from a branch, branch main, pasta /root. Pronto, você tem uma URL pública.
